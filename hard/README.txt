This game modifies the version 0, and is version 2 of the tic-tac-toe game application. 

The changes are made in the javascript file, which now uses a minimax algorithm, as described at  https://www.freecodecamp.org/news/how-to-make-your-tic-tac-toe-game-unbeatable-by-using-the-minimax-algorithm-9d690bad4b37/.

This is the hard level of the game in which the ai does not let the player win. There can only be two outcomes, as opposed to three outcomes in easy mode (version 0),and  possibly three outcomes in medium mode(version 1).
